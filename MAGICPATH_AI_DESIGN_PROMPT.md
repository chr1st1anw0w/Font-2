# 動態紋理生成工具 - MagicPath.ai 設計提示詞

**用途**: 用於 MagicPath.ai 生成完整的 UI 設計稿
**應用名稱**: Dynamic Texture Generator
**設計風格**: 現代深色主題 + 紫色科技感
**目標用戶**: 設計師、藝術家、開發者

---

## 🎨 整體設計概述

### 應用架構
```
一個完整的紋理生成應用，包含四個主要界面區域：
- 頂部: 工具欄 (TopBar)
- 中央: Canvas 繪製區域 (800×800px)
- 右側: 參數控制面板 (TextureControlPanel)
- 底部: 浮動工具欄 (TextureToolbar、NavigationBar)
```

### 設計基調
- **主色系**: 深灰色 (#0d1117, #1a202c, #111827) + 紫色科技感 (#7c3aed, #5847eb)
- **文本顏色**: 淺灰色 (#d1d5db, #9ca3af) 適合深色背景
- **邊框**: 深灰色邊框 (#374151, #4b5563)
- **交互反饋**: 紫色高亮 + 漸層效果
- **動畫**: 平滑過渡 (200-300ms)

## 🎯 設計要點總結

1. **深色主題**: 所有背景都使用深灰色系，強調紫色交互元素
2. **層次清晰**: 使用不同深度的灰色區分主次區域
3. **紫色科技感**: 所有交互元素使用紫色 (#7c3aed) 高亮
4. **平滑動畫**: 所有動畫使用 200ms 左右的平滑過渡
5. **實時反饋**: 滑桿拖動時實時顯示數值
6. **模態設計**: 模態使用半透明黑色背景 (bg-black/50)
7. **圓角一致**: 主要元素 8px 圓角，按鈕 6px 圓角
8. **間距規範**: 使用 8px、16px、24px 的倍數間距
